# 🌐 Micro Concept: Day 1

```
TOPIC: What is Cybersecurity? (30 seconds)
══════════════════════════════════════════

Cybersecurity = Protecting computers, servers, networks,
and data from digital attacks.

3 Pillars (CIA Triad):
  C = Confidentiality (Only authorized can access)
  I = Integrity       (Data is not tampered)
  A = Availability    (System is always accessible)

Common Threats:
  → Phishing (fake emails)
  → Malware (viruses)
  → SQL Injection (attacking databases)
  → DDoS (overwhelming servers)
```
